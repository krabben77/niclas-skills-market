# Obsidian Assistant

Teknisk assistent för Obsidian-vaultets Claude-system och Obsidians egna plugin-API.

## Skills

### `/vault-assistant`
Hjälper dig förstå, bygga och ändra vaultets Claude Code-system:
hooks, scripts, vault-manifest, frontmatter-validering och behörighetsnivåer.

### `/obsidian-dev`
Teknisk hjälp med Obsidians plugin-API, Community Plugins, Dataview och Bases.
Hämtar alltid aktuell dokumentation från officiella källor.

### `/debug-system`
Strukturerad felsökning av både vault-systemet och Obsidians egna system.
Tar dig från symptom till rot-orsak och åtgärd.
